## 2. Ch6 — Controllability & Observability (가제어성·가관측성)

### 2.1 직관 (교수 비유)
- **Controllable**: 시스템 상태를 입력 $u$로 원하는 곳으로 옮길 수 있나? → *"자동차에 핸들 없이 운전 가능?"* 제어기 설계의 전제.
- **Observable**: 출력 $y$만 보고 내부 상태(초기치)를 알아낼 수 있나? → *"범죄 현장 증거로 그때 상황 추정"*. 필터/추정기와 연결.
- 입력 크기 제한 없음, 유한시간 안에 옮기기만 하면 됨 (*"부산→서울, 인천 들렀다 와도 됨. 단 유한시간"*).

### 2.2 Controllability 판정 (동치 조건들)
$\dot{\mathbf x}=A\mathbf x+B\mathbf u$, $A\in\mathbb R^{n\times n}$.

1. **Controllability matrix** full rank:
$$\mathcal C=[\,B\ \ AB\ \ A^2B\ \cdots\ A^{n-1}B\,],\qquad \rho(\mathcal C)=n$$
2. **Gramian** nonsingular (어떤 $t>0$):
$$W_c(t)=\int_0^t e^{A\tau}BB^Te^{A^T\tau}\,d\tau \succ 0$$
   → $W_c$의 inverse가 있어야 $x_0\to x_1$ 보내는 입력이 존재 (책의 "2초 만에 0으로 보내기" 예제).
3. **PBH test**: 모든 eigenvalue $\lambda$에 대해
$$\rho([\,A-\lambda I\ \ B\,])=n$$
   → **각 eigenvalue(모드)별로** controllable한지 알 수 있는 게 장점. 어떤 모드가 안 움직이는지 콕 집어냄.
4. (대각화 가능 시) 모든 mode가 입력과 결합돼 있음.

### 2.3 Observability 판정 (쌍대)
$\dot{\mathbf x}=A\mathbf x$, $y=C\mathbf x$.

1. **Observability matrix** full rank:
$$\mathcal O=\begin{bmatrix}C\\CA\\CA^2\\\vdots\\CA^{n-1}\end{bmatrix},\qquad \rho(\mathcal O)=n$$
2. Gramian $W_o(t)=\int_0^t e^{A^T\tau}C^TCe^{A\tau}d\tau\succ0$.
3. **PBH**: 모든 $\lambda$에서 $\rho\!\begin{bmatrix}A-\lambda I\\C\end{bmatrix}=n$.

**쌍대성(Duality)**: $\{A,B\}$ controllable $\iff$ $\{A^T,B^T\}$ observable. → observability는 controllability 결과를 전치해 그대로 사용.

### 2.4 중요 성질
- **Similarity transform 불변**: $\bar A=PAP^{-1},\bar B=PB$ 변환해도 controllability 안 바뀜. ($\rho$ 보존, $P$ nonsingular). → 상태변수 조합만 바꾸는 거라 제어가능성 본질은 그대로 (0428 후반 정리 6.2·6.3).
- **Controllability index** $\mu$: $[\,B\ AB\cdots A^{\mu-1}B\,]$가 rank $n$ 되는 최소 $\mu$. 다입력일 때 $A^{n-1}B$까지 안 가도 됨. $\mu \le n-p+1$ ($p=\rho(B)$).
- **Stabilizable**: unstable mode가 모두 controllable (안정화만 가능하면 됨). **Detectable**: unstable mode가 모두 observable.

### 2.5 예상 출제 유형
- ★ **2×2 또는 3×3 시스템 controllability/observability 판정** ($\mathcal C,\mathcal O$ rank 또는 det).
- ★ **Stabilizable/Detectable 개념 설명 + 판정** (족보 2017 p3-1 그대로: $\dot x=\begin{bmatrix}0&1\\0&-3\end{bmatrix}x+\begin{bmatrix}1\\0\end{bmatrix}u,\ y=[0\ 1]x$).
- PBH로 어떤 eigenvalue가 uncontrollable한지 찾기.

## 3. Ch7 — Minimal Realization & Coprime Fractions (최소실현·서로소분수)

### 3.1 핵심 개념
- **Minimal realization**: 전달함수 $G(s)$를 실현하는 $(A,B,C,D)$ 중 **차수가 최소**인 것. ⟺ $\{A,B\}$ controllable **그리고** $\{A,C\}$ observable.
- 차수 = $G(s)$를 기약분수로 줄였을 때 분모의 차수.
- **Coprime (서로소)**: 두 다항식 $N(s),D(s)$가 **공통근(common root)이 없음**. → 약분되는 pole-zero가 없음.
  - 판정: 공통인수 없음 ⟺ 두 다항식의 **Sylvester resultant ≠ 0**, 또는 **Bezout 항등식** $A(s)N(s)+B(s)D(s)=1$을 만족하는 $A,B$ 존재.
  - 족보 p3-4: $G=\frac1{s+1}$, $H=\frac{(s+3)(s+4)}{(s+1)(s+2)}$ coprimeness 확인 → "no unstable common zero → coprime".

### 3.2 왜 중요한가
- pole-zero cancellation이 있으면 (특히 unstable한 게 약분되면) → realization이 minimal 아님 + total stability 깨짐(Ch9와 연결).
- Ch9 극배치의 해 존재 조건이 **$D(s),N(s)$ coprime**.

### 3.3 예상 출제 유형
- 두 전달함수/다항식의 **coprimeness 판정** (15점, 족보 패턴).
- $G(s)$의 minimal realization 차수 묻기.

## 4. Ch8 — State Feedback & State Estimators (상태궤환·상태추정기)

### 4.1 State Feedback 극배치
- 제어법칙: $\mathbf u = r - K\mathbf x$ (또는 $u=pr-Kx$). 폐루프: $\dot x=(A-BK)x+Br$.
- **{A,B} controllable ⟺ $A-BK$의 eigenvalue를 임의로 배치 가능** (핵심 정리).
- **단일입력 절차**: 원하는 특성다항식 $\Delta_d(s)$의 계수와 $\det(sI-(A-BK))$ 계수를 매칭 → $K$ 연립으로 풀이.
  - 족보 p3-1(4): $A+BF=\begin{bmatrix}f_1&1+f_2\\0&-3\end{bmatrix}$ 꼴로 전개해 $f_1,f_2$ 결정.

### 4.2 Lyapunov 방정식 방법 (0519 — 다입력 가능)
- 원하는 eigenvalue를 가진 $F$ 선택 → **Sylvester 방정식** $AT-TF=B\bar K$ 풀어 $T$ → $K=\bar K T^{-1}$.
- $A-BK = TFT^{-1}$ (similar) → eigenvalue = $F$의 eigenvalue.
- **조건**: $F$의 eigenvalue가 $A$의 eigenvalue와 **겹치면 안 됨** (해 존재). 겹치면 살짝 옮김(1→1.01).
- $T$가 singular면 $\bar K$ 다시 잡아 재시도. "$\{A,B\}$ controllable & $\{A,F\}$ observable이면 nonsingular $T$" 는 **necessary 조건이지 sufficient 아님** (교수 반복 강조).
- 장점: 다항식 풀이보다 대수방정식이라 차수 커도 컴퓨터로 풀기 쉬움.

### 4.3 Feedforward gain & Tracking
- **Regulation**: $r=0$, 상태/에러를 0으로. **Tracking**: step 같은 특정 입력 추종. **Servomechanism**: 임의 입력 추종(*"주인이 여러 개"*).
- **Feedforward gain $p$**: $u=pr-Kx$에서 정상상태에 $y(\infty)=r$ 되도록 폐루프 DC gain을 1로 맞춤 → $p=1/G_{cl}(0)$.
- **Final Value Theorem**: $x(\infty)=\lim_{s\to0}sX(s)$, $x(0^+)=\lim_{s\to\infty}sX(s)$. step 입력 $=a/s$.
- 입력 saturation 주의 (*"에일러론 ±15°인데 +60° 명령하면 포화"*) → $K$ 너무 크게 잡으면 안 됨.

### 4.4 Robust Tracking & Disturbance Rejection (0512)
- 플랜트에 disturbance $w$: $\dot x=Ax+Bu+Bw$.
- **에러를 상태로 augment**: $\dot x_a=r-y=r-Cx$ → 적분제어 추가, $u=Kx+K_a x_a$.
- **Theorem 8.5**: 원 플랜트 controllable + **플랜트가 $s=0$에 zero 없음** ⟹ augmented plant도 controllable → 극 임의배치 가능. ("no zero at $s=0$" 조건 추가)
- augmented 특성방정식 = $s\bar D(s)+K_a\bar N(s)$ 형태 → **교수가 "이거 기억해야 된다"고 강조한 키 포인트**.

### 4.5 State Estimator / Observer (0519)
- **Full-order**: $\dot{\hat x}=(A-LC)\hat x+Bu+Ly$. 추정오차 $e=x-\hat x$ → $\dot e=(A-LC)e$ → **입력 $u$와 무관, $x$와 decouple**. $A-LC$ eigenvalue를 LHP로 배치하면 $e\to0$.
- **{A,C} observable ⟺ $A-LC$ eigenvalue 임의배치 가능**.
- **Estimator pole은 제어 pole보다 훨씬 왼쪽에** (빨리 수렴). *"estimator는 소프트웨어 계산만 하니 $L$ 크게 해도 비용 없음. 제어 $K$는 크면 saturation"*.
- **Reduced-order**: 출력 $q$개면 차원 $n-q$로 축소. $F$는 $(n-q)$차 stable, eigenvalue가 $A$와 안 겹쳐야 함.
- **Separation Principle**: 추정상태로 궤환해도 (estimator error)와 (controller)가 **분리**됨 → 전체 closed-loop pole = {제어 pole} ∪ {estimator pole}. 전달함수는 estimator 없는 것처럼.

### 4.6 예상 출제 유형
- ★ **2×2 state feedback으로 원하는 pole 만드는 $K$(또는 $F$) 구하기** (족보 p3-1(4) 그대로).
- ★ **Observer gain $L$ 설계** ($A-LC$ 특성다항식 매칭).
- Robust tracking 구조에서 augmented 시스템 세우기, Theorem 8.5 조건.
- separation principle / necessary vs sufficient 개념 설명(단답).

## 5. Ch9 — Pole Placement & Model Matching via Coprime Fractions (극배치·모델매칭)

### 5.1 Coprime Fraction 극배치 (★ 최우선 출제 예상)
- 플랜트 $G(s)=\dfrac{N(s)}{D(s)}$, 제어기 $C(s)=\dfrac{B(s)}{A(s)}$.
- 폐루프 특성다항식: $$F(s)=D(s)A(s)+N(s)B(s)$$ → 원하는 극의 다항식과 같게.
- **Sylvester resultant matrix** $S_m$ 구성: $D,N$ 계수를 한 칸씩 shift하며 배열, $S_m\cdot[\text{제어기 계수}]=[\,F\text{ 계수}\,]$ → **중학생도 푸는 연립방정식**(교수 표현).
- **해 존재 조건 (Theorem 9.M)**: $S_m$ full column rank → **제어기 차수 $m\ge n-1$** ($n=$플랜트 차수). $D,N$ coprime이면 $m=n-1$에서 항상 유일해.

#### ★ 핵심 예제 (교수가 0526·0528 두 번 풀이 — 거의 확정)
플랜트 $G(s)=\dfrac{s-2}{s^2-1}$ ($n=2$). 원하는 극 $\{-2,\,-1\pm j\}$ 류.
- $m=n-1=1$ → $A(s)=a_0+a_1s,\ B(s)=b_0+b_1s$.
- $D(s)=-1+0\cdot s+1\cdot s^2,\ N(s)=-2+s$.
- $S_m$ (4×4) 세워 $a_0,a_1,b_0,b_1$ 풀이 → 제어기 $C(s)=B(s)/A(s)$.
- $G(0)$로 tracking 확인.

### 5.2 Robust Tracking & Internal Model Principle (0528)
- $\phi(s)$ = 레퍼런스 $r$과 disturbance $w$의 **unstable pole들의 LCM**. (step이면 $1/s$ → $\phi(s)=s$; $\sin\omega t$면 $s^2+\omega^2$.)
- 제어기 분모에 $\phi(s)$ 포함: $C(s)=\dfrac{B(s)}{A(s)\phi(s)}$ → **Internal Model Principle** (PID 적분기 원리).
- **Corollary 9.3 조건**: **$\phi(s)$의 근이 $N(s)$의 zero가 아님** (겹치면 cancellation으로 추종 실패) + $N,D$ coprime.
- 두 구현법: ① $\phi$ 분리해 $(B/A)(1/\phi)$, ② 제어기 차수 1 높여 free parameter 만들고 **$a_0=0$ 강제**(적분기 자연 포함) → $S_m$ 축소(6×6 → 6×5)돼 더 쉬움.
- **수치 직관**: 플랜트가 변동되면(예 $\frac{s-2.1}{s^2-0.95}$) $G(0)\approx0.7875$ → step의 79%만 추종, **20% 추종오차** → robust 설계 필요성. (★ 인상적 예제, 출제 가능)

### 5.3 Model Matching (0528)
- 극배치는 pole만, **model matching은 pole+zero 둘 다** 목표모델 $G_o(s)$에 맞춤. $r\to y$ 전달함수 $=G_o(s)$.
- $G_o(s)=1$이면 best possible (*"조종간 착 하면 비행기도 착"*). 단 $r$과 $y$는 power level 다름.

### 5.4 Implementable / Well-posed / Total Stability (개념 출제 유력)
- **Implementable transfer function 3조건**:
  1. **Proper**: 제어기 분모차수 ≥ 분자차수 (미분 필요 X, noise 취약 X).
  2. **No leakage**: $r\to y$가 반드시 플랜트를 통과(입력 $u$ 거쳐야). *"군대 안 왔는데 보고만 한 격, 눈 가리고 아웅"*.
  3. **Total stability**: 모든 input-output pair 전달함수가 BIBO stable.
- **Well-posed**: 모든 input-output pair 전달함수가 proper. 조건 $G(\infty)C(\infty)\ne-1$. (strictly proper면 $G(\infty)=0$ → 무조건 OK.)
- **Total stability**: $r\to y$ BIBO stable + **제어기·플랜트 사이 unstable pole-zero cancellation 없음**.
  - 예(Fig 9.3): $s-2$(unstable pole)가 약분되면 $r\to y$는 안정해 보여도 noise→$y$ 전달함수에 $s-2$ 남아 **발산**.

### 5.5 Mason's Gain Rule (보조 도구)
- $T=\dfrac{\sum_k P_k\Delta_k}{\Delta}$, $\Delta=1-\sum(\text{loop gain})+\sum(\text{non-touching 2개 곱})-\cdots$. 블록선도 전달함수 유도용.

### 5.6 예상 출제 유형
- ★★★ **Coprime fraction으로 제어기 $C(s)=B/A$ 계수 구하기** (Sylvester matrix, $G(s)=\frac{s-2}{s^2-1}$ 류). 거의 확정.
- ★ **Internal model 포함 robust tracking 제어기 설계** ($\phi(s)=s$ 등).
- ★ **개념 정의 문제**: well-posed / total stability / implementable 3조건 / internal model principle / regulator vs tracking vs servo.
- coprimeness 판정, model matching 개념, Mason rule.
