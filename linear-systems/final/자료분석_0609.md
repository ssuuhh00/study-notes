## 2. 친구 손글씨 요약 (선시기말범위.pdf) — 판독·정리

### 2-1. Ch6 (가제어성·가관측성)
- Controllability / Observability **의미 + 체크법**, controllability/observability **index**.
- ★ **Decomposition** (4분할): 시스템을 닮음변환으로 아래 4부분으로 쪼갬 — **각 부분이 입력 u / 출력 y에 연결되는지**가 핵심.

  | 부분 | u에 연결? | y에 연결? | 의미 |
  |---|---|---|---|
  | **co** (controllable & observable) | ✓ | ✓ | u로 움직이고 y로 보임 — 전달함수에 나타나는 부분 |
  | **cō** (controllable, **un**observable) | ✓ | ✗ | u로 움직이지만 y엔 안 보임 |
  | **c̄o** (**un**controllable, observable) | ✗ | ✓ | u로 못 움직이지만 y로 보임 |
  | **c̄ō** (둘 다 ✗) | ✗ | ✗ | u·y 둘 다 단절 |

  (손글씨 표기 그대로: `cō: u연결 y연결X`, `c̄o: u연결X y연결`, `co: ○○`, `c̄ō: ××`)
  → 전달함수 $G(s)=C(sI-A)^{-1}B$ 는 **co 부분만** 반영. (uncontrollable/unobservable은 약분돼 사라짐 = minimal realization과 연결.)

- ★ **Jordan form으로 c/o 판정**: 같은 eigenvalue $\lambda$의 **Jordan block이 여러 개**일 때, 각 block의
  - **controllability** ← (그 block들에 대응하는 **B의 마지막 행**들이 서로 **independent**한가)
  - **observability** ← (대응하는 **C의 첫 열**들이 independent한가)
  - (녹취 line 40–45: "조단 블락이 몇 개 있나 … 인디펜던트 해야 되고 … 람다2에 대해선 하나밖에 없네 → 이거만" — block 개수 세고 independence 확인하는 절차)

### 2-2. Ch8 (상태궤환·상태추정기)
- **State feedback**: x 전부 측정 가능 가정, $u=-Kx$(+r). **Estimator**: u,y로 x 추정해서 그 $\hat x$로 궤환.
- **Eigenvalue assignment**: {A,B} controllable ⟺ $A-BK$ 극 임의배치. complex는 **켤레쌍**으로. **state feedback은 pole만 옮기고 zero는 안 바뀜.**
- 극 배치 요령: **왼쪽**에, 너무 **붙이면 X**(이득↑·반응↓), **등간격**이 좋음.
- ★ **Lyapunov(Sylvester) 방정식 풀이** (다입력 가능):
  1. 원하는 eigenvalue 가진 **F 설계**
  2. **K̄** 잡되 $\{F,\bar K\}$ observable
  3. **$AT-TF=B\bar K$** 풀어 **T** 구함
  4. **$K=\bar K T^{-1}$**
  - 조건: **F의 eigenvalue가 A의 eigenvalue와 겹치면 안 됨**(겹치면 $-1\to-1.01$로 살짝 이동). T nonsingular은 $\{A,B\}$ cont. & $\{F,\bar K\}$ obs.일 때 ← **necessary, not sufficient**.
- **Regulation**(r=0, 출력→0) / **Tracking**(step 추종) / **Servomechanism**(임의 r(t) 추종). feedforward gain $p=1/G_{cl}(0)$, **Final Value Thm**.
- **Robust tracking & disturbance rejection**: 파라미터 변동·외란에도 추종 → **적분기(1/s)** 추가 = **Internal Model**(외부신호의 unstable part를 제어기가 내장).
- **Estimator**: 오차 $e=x-\hat x$, $\dot e=(A-LC)e$ (입력 무관). {A,C} observable ⟺ $A-LC$ 극 배치. **Reduced-order**(출력 개수만큼 차수 down). **Separation Principle**: estimator·controller 독립 설계, 전체극 = 제어극 ∪ estimator극.

### 2-3. Ch9 (극배치·모델매칭, 전달함수)
- **Diophantine / pole placement**: 제어기 $C=\dfrac{B(s)}{A(s)}$, 폐루프 특성식 $A(s)D(s)+B(s)N(s)=F(s)$.
  - $D,N$ **coprime**이면 $\bar A D+\bar B N=1$ 인 $\bar A,\bar B$ 존재 → 특수해 $A=F\bar A,\ B=F\bar B$.
  - 일반해 = 특수해 + (동차해 $\hat A D+\hat B N=0$, 즉 $\hat A=N,\hat B=-D$에 임의 $Q$) → $Q$ 잘 골라 차수 최소화.
  - 실전은 그냥 **계수비교(Sylvester resultant $S_m$)** 로 연립 풀이. **미지수=제어기 계수($A,B$), 나머지(D,N,F)는 알고 있음.**
  - 차수: m 1↑ → 미지수 2↑·식 1↑ → **자유도(free parameter) 1↑**. 해 존재 최소차수 **$m\ge n-1$**.
- ★ **Robust tracking — Internal Model**: $\phi(s)=$ (r·w의 **unstable pole들의 LCM**). step→$\phi=s$, $\sin\omega t\to s^2+\omega^2$.
  - 설계: $1/\phi$를 **플랜트에 포함시켜** 제어기 설계 → 구현 시 $C=\dfrac{B}{A\phi}$로 정리. 그러면 r·w의 unstable 성분이 **제어입력(u/l)단에서 상쇄**(y가 아니라).
- ★ **Embedding internal model (적분기 자동 내장)**: 제어기 $A(s)=a_0+a_1s+\cdots$ 에서 **$a_0=0$** 강제 → $1/s$ 적분기가 자동 포함.
  - 단, $a_0=0$ 제약을 넣으려면 **자유도 1 필요** → m을 **$n-1\to n$** 로 하나 올림. (n−1이면 유일해라 제약 못 넣음.)
- ★ **Implementable = Well-posed + Totally stable** (개념).
  - **Well-posed**: 모든 입출력쌍 전달함수가 proper. ($G(\infty)C(\infty)\ne-1$.)
  - **Totally stable**: 모든 입출력쌍 BIBO stable = 제어기·플랜트 사이 **unstable pole-zero 약분 없음**.
- ★ **Two-parameter controller (model matching 구현)**:
  $$C_1=\frac{L(s)}{A(s)},\quad C_2=\frac{M(s)}{A(s)},\quad \hat g_o=\frac{LN}{AD+MN}=\frac{E}{F}.$$
  - 절차: 목표 $\hat g_o$ 주어짐 → $N$으로 나눠 $E,F$ 잡고 → **$A,L,M$** 풀이. (분모 $AD+MN=F$ 는 pole placement처럼, 분자 $LN=E$ 는 model matching.)
  - 교재 **Example 9.9**(KakaoTalk 손풀이 자료)가 이 two-parameter 구현 예제.

## 3. 족보 (선시 기말.pdf) — 2013·2014·2015 기말

> ⚠️ 2013·2014엔 **중간범위(convolution·Cayley-Hamilton·realization·BIBO)** 문제도 섞여 있음 → **이번 기말(6·8·9장)엔 해당 안 됨.** 아래는 **기말범위 문제만** 추림.

### 2013 (6/20) — ★ 거의 그대로 나올 패턴
- **Q2.** $\dot x=\begin{bmatrix}-1&3\\3&-1\end{bmatrix}x+\begin{bmatrix}1\\1\end{bmatrix}u,\ y=[1\ 1]x$ → **controllable form으로 reduce**(uncontrollable 부분 제거), reduced eq이 observable인가? → **Ch6 decomposition 직결**
- **Q3.** $\dot x=\begin{bmatrix}-1&1\\0&1\end{bmatrix}x+\begin{bmatrix}0\\1\end{bmatrix}u,\ y=[1\ 0]x$:
  - a) **state feedback**으로 극 $-2,-3$ 배치 (K)
  - b) **tracking**(state feedback + feedforward gain), step 추종
  - c) **observer** 극 $-6,-10$ 설계 (L)
  - d) **output feedback**으로 극 $-2,-3$ (전달함수 제어기)
  - e) a)와 d) 결과 비교·논의

### 2014 (6/17) — 기말범위만
- **Q4.** $A=\begin{bmatrix}0&0&-2\\0&1&0\\1&0&3\end{bmatrix},B=\begin{bmatrix}0\\1\\0\end{bmatrix},C=[1\ 0\ 0]$ → (a) controllability (b) observability (의미 설명 + 판정)

### 2015 (6/15) — ★ 전부 기말범위, 개념서술형 多
- **Q2.** $\dot x=\begin{bmatrix}1&1&0\\0&1&0\\0&1&2\end{bmatrix}x+\begin{bmatrix}0\\1\\0\end{bmatrix}u,\ y=[1\ 0\ 0]x$ → observability·controllability 의미 + 판정 (Jordan-ish 3×3)
- **Q3.** coprime 의미 + $\hat g(s)=\dfrac{s+1}{4(s^2+s+1)}$ coprimeness 판정
- **Q4.** **Lyapunov 방정식으로 eigenvalue 배치하는 방법** 설명
- **Q5.** **state estimator 일반구조 + estimator 극 배치 방법** 설명
- **Q6.** **disturbance 있는 robust tracking** 얻는 방법 설명

→ **2015 Q4·Q5·Q6 = 개념서술형**이 그대로 출제 가능성 높음(녹취 강조와 일치).
